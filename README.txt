Baldax Bros
==========
"Baldax Bros" est un jeu inspiré par les classiques du genre plateforme, développé dans le cadre de ma formation en Développement Web et Web Mobile. 
Ce projet utilise HTML, CSS, et JavaScript pour créer une expérience ludique rappelant les aventures de Mario, mais avec une touche personnelle.


Fonctionnalités
---------------
Personnages Animés : Mario, Peach, et un monstre, chacun intégré avec des animations spécifiques.
Obstacles et Interactions : Gestion des collisions avec les obstacles et animations spécifiques en cas de rencontre avec des éléments du jeu.
Contrôles Intuitifs : Déplacement du personnage principal avec les flèches directionnelles et saut avec la barre d'espace.
Affichage Dynamique : Informations sur le jeu affichées en temps réel, y compris un compteur de temps.

Technologies Utilisées
----------------------
HTML5 pour la structure du jeu.
CSS3 pour le stylisme et les animations.
JavaScript pour la logique du jeu, les contrôles des personnages et la gestion des collisions.

Installation
------------
Pour lancer "Baldax Bros" rendez-vous sur github.

Jouer
-----
Utilisez les touches suivantes pour contrôler le personnage principal :
Flèche gauche : Déplacer à gauche.
Flèche droite : Déplacer à droite.
Barre d'espace : Sauter
